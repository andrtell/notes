# Debug using Chrome

Start node in inspect mode.

```sh
$ node --inspect-brk script.js

Debugger listening on ws://127.0.0.1:9229/695636d0-6874-4673-802b-2dc860101504
```

Open Chrome. Visit: `chrome://inspect`.
